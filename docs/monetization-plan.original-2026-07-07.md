# Saajuu 수익화 구체화 계획

작성일: 2026-07-07
현재 상태: Vite 정적 웹앱 (v0.2.0.0), 브라우저 내 `manseryeok` 기반 사주 계산 + 규칙 기반 상세 풀이, GitHub Pages 배포. 백엔드·계정·결제·AI 없음.

---

## 1. 시장 배경 (조사 요약)

- 국내 점술 시장 규모 약 **1조 4천억 원** 추산. 비대면·앱 기반으로 빠르게 이동 중.
- **점신(테크랩스)**: 2024년 매출 830억 원, 영업이익 100억 원 (전년 대비 매출 +58%).
- **포스텔러(운칠기삼)**: 누적 가입자 860만 명, 연 매출 100억 원 돌파, 2023년 흑자전환.
- 지배적 수익 모델: **프리미엄(freemium)** — 기본 운세 무료, 상세·맞춤 해석 유료.
  - 가격대: 990원 소액 단건 ~ 10만 원대 종합 패키지.
- AI 사주 결제 전환의 핵심: "동일 AI라도 데이터 구성·프롬프트 설계·해석 방식이 결제 여부를 가른다."
  점집 기본 상담료 5만 원+ 대비 AI는 저렴하면서 만족도가 높다는 인식이 확산 중.

시사점: 광고만으로는 트래픽 없이는 수익이 미미하다. **핵심 수익원은 "유료 상세 풀이" 단건 결제**이고, 광고·제휴는 보조 수단이다.

## 2. 수익 모델 옵션 비교

| 모델 | 초기 비용 | 필요 인프라 | 예상 마진 | 판단 |
|------|----------|------------|----------|------|
| ① 디스플레이 광고 (애드센스/카카오 애드핏) | 도메인비 정도 | 없음 (정적 유지) | 트래픽 의존, 초기 월 수천~수만 원 | 보조 수단. 즉시 착수 가능 |
| ② 제휴 (쿠팡파트너스 — 개운 아이템 등) | 0원 | 없음 | 판매액의 ~3% | 결과 페이지에 자연스럽게 삽입 가능 |
| ③ **AI 프리미엄 풀이 단건 결제** | 사업자등록 + PG 계약 | 백엔드(Supabase) + LLM API + 토스페이먼츠 | **원가 100~500원/건, 판매 3,900원 → 마진 85%+** | **본명 수익원. Phase 1 목표** |
| ④ 구독 (월간 운세, 궁합 등) | ③ 이후 확장 | ③과 동일 + 계정 | 높음 | Phase 2 |
| ⑤ 광고형 리워드 (광고 보면 무료 1회) | ③ 이후 | 광고 SDK | 중간 | 선택 사항 |

## 3. 단계별 로드맵

### Phase 0 — 기반 다지기 (1~2주, 사업자 없이 가능)

목표: 트래픽·전환 퍼널을 만들고 광고 심사를 준비한다.

1. **커스텀 도메인 구입 및 연결** (예: saajuu.kr, 연 2만 원 내외)
   - GitHub Pages는 커스텀 도메인 지원. 애드센스 승인에도 유리.
2. **공유 기능**: 결과를 이미지 카드로 저장/공유 (카카오톡 공유 버튼). 사주 서비스는 공유가 곧 마케팅이다.
3. **SEO 콘텐츠 페이지**: "오늘의 운세", "띠별 운세", "일주별 성격" 등 정적 콘텐츠 페이지 추가
   → 검색 유입 + 애드센스 승인용 콘텐츠 볼륨 확보.
4. **애드센스 + 카카오 애드핏 신청**, 쿠팡파트너스 가입 (개운 아이템·책 추천을 결과 하단에 배치).
5. **전환 실험 (가짜 문 테스트)**: "프리미엄 상세 풀이 보기 (준비 중)" 버튼을 두고 클릭률 측정
   → 실제 결제 수요를 돈 들이기 전에 검증.

### Phase 1 — AI 프리미엄 풀이 + 결제 (4~8주)

목표: 첫 매출. 무료(현재 기능) → 유료(AI 심층 풀이) 퍼널 완성.

**상품 설계**
- 무료: 사주팔자 + 오행 분석 + 요약 풀이 (현재 기능, 미끼 상품)
- 유료 단건: **"AI 심층 사주 리포트" 3,900원** — 성격·직업·재물·연애·건강 + 대운 흐름, A4 5~8장 분량
- 시즌 상품: 신년운세(연말 트래픽 폭증 — 밤 12시 트래픽 폭주 기사 참고), 궁합 리포트 4,900원

**기술 구조**
- 백엔드: **Supabase Edge Functions** (무료 티어로 시작 가능)
  - 프론트에서 사주 차트 계산(기존 코드 재사용) → 서버에서 LLM 호출 → 리포트 생성
  - API 키를 클라이언트에 노출하지 않기 위해 서버 필수
- LLM: Claude API (claude-haiku-4-5 시작, 품질 필요 시 claude-sonnet-5)
  - 원가 추정: 리포트 1건당 약 50~500원 (모델에 따라) → 3,900원 판매 시 마진 85% 이상
  - 차별화 포인트: 이미 만든 **근거 기반(evidence-backed) 풀이 데이터**를 프롬프트에 주입 → "그럴듯한 헛소리"가 아닌 만세력 근거가 붙은 해석
- 결제: **토스페이먼츠** (개인사업자 가능, 개발 문서 우수, 수수료 약 3% 내외)
  - 단건 결제만 먼저. 결제 완료 → 리포트 생성 → 이메일/링크 전달 (계정 없이 시작 가능)

**행정 준비 (필수)**
1. ⚠️ **겸업 규정 확인** — 재직 중이므로 회사 취업규칙의 겸업 금지/신고 조항 최우선 확인
2. 개인사업자 등록 (간이과세, 홈택스에서 무료·당일 가능. 업종: 응용소프트웨어 개발 또는 정보서비스)
3. 통신판매업 신고 (카드 결제 필수 요건. 간이과세자는 면제 대상 여부 확인 — 면제여도 PG 계약에는 필요할 수 있음)
4. 이용약관·환불규정·개인정보처리방침 작성 (생년월일 수집 = 개인정보 → 수집·이용 동의 필수. 서버에 저장하지 않고 즉시 폐기하는 구조로 설계하면 부담 최소화)

### Phase 2 — 확장 (매출 발생 후)

- 월 구독제 (월간 운세 + 무제한 질문 채팅) 6,900~9,900원
- 궁합·택일·작명 등 상품 라인 확장
- 앱 스토어 진출 (Capacitor로 기존 웹앱 래핑) — 인앱결제 수수료 15~30% 유의
- 리텐션: 매일 오는 "오늘의 운세" 알림 (웹 푸시)

## 4. 비용·수익 추정

**고정비 (월)**
- 도메인: ~2,000원/월 환산
- Supabase: 무료 티어 → 유료 전환 시 $25/월
- 합계: 초기 월 5천 원 미만

**변동비**: LLM 호출 ~50–500원/건 + PG 수수료 ~3.3%

**손익분기**: 사실상 리포트 월 10건 판매면 고정비 회수.

**시나리오 (단가 3,900원 기준)**
| 시나리오 | 일 방문자 | 유료 전환율 | 월 매출 |
|---------|----------|------------|---------|
| 보수적 | 100명 | 1% | 약 12만 원 |
| 기본 | 500명 | 2% | 약 117만 원 |
| 신년 성수기 | 3,000명 | 3% | 약 1,000만 원+ |

핵심 변수는 단가가 아니라 **트래픽**이다. 12월~2월(신년운세 시즌)이 이 시장의 절대적 성수기이므로, **늦어도 11월까지 Phase 1을 완성**하는 것이 최적 타이밍이다.

## 5. 리스크

1. **트래픽 확보 실패** — 가장 큰 리스크. 대응: 공유 기능 + SEO + 시즌 타이밍. Phase 0의 가짜 문 테스트로 수요 검증 후 투자.
2. **겸업 규정 위반** — 착수 전 반드시 확인. 배우자 명의 사업자 등 대안 검토 가능.
3. **대형 경쟁사** — 포스텔러·점신과 정면 승부 불가. 틈새: 앱 설치 없는 웹 기반 + 근거 기반 풀이 + 특정 세그먼트(예: 커리어 사주) 집중.
4. **LLM 품질 불만/환불** — 결제 전 샘플 리포트 일부 공개, 명확한 환불 규정으로 대응.
