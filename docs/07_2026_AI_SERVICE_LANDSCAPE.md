# 2026 AI 서비스 지형 분석

작성일: 2026-07-19  
목적: 최근 빠르게 늘어난 AI 관계/상담/점성술 서비스의 구조를 정리하고, Saajuu에 실제로 반영할 항목만 남긴다.

---

## 1. 요약

지금 시장은 크게 세 갈래로 갈린다.

1. **관계 유지형 앱**
   - 커플이 매일 같이 들어오게 만드는 구조
   - 예: Flamme, Maia, Arya
2. **자기이해형 점성술 앱**
   - 차트보다 관계, 성격, 타이밍 해석을 앞세움
   - 예: The Pattern, Co-Star, Aistro
3. **AI 감정 정리/저널링 앱**
   - 대화를 누적 기억하고, 이전 맥락을 이어서 확장
   - 예: Rosebud

Saajuu는 이 셋을 억지로 다 하는 서비스가 아니라, 아래 조합으로 가는 편이 맞다.

> 한국식 사주 계산 근거  
> + 관계/결정 고민 중심 무료 콘텐츠  
> + 기억이 이어지는 상담형 UX  
> + 로그인 후 인연 연결과 재방문 루프

## 3. 지금 시장에서 검증된 패턴

### 3.1 검증된 것

1. **관계 서비스는 재방문 구조가 있어야 유지된다.**
   - 커플 연결
   - 일일 질문
   - 마일스톤
   - 작은 행동 제안

2. **AI 상담은 단발 답변보다 맥락 기억이 중요하다.**
   - 사용자는 같은 말을 다시 쓰는 것을 싫어한다.
   - 매 턴 요약과 다음 질문 설계가 있어야 한다.

3. **전문 용어를 숨기고 생활 언어로 번역하는 앱이 강하다.**
   - "십신/오행" 설명 자체보다
   - "지금 관계에서 왜 속도 차이가 나는지"
   - "왜 혼자 밀어붙이면 리스크가 커지는지"
   - 같은 식의 번역이 중요하다.

4. **강한 톤앤매너는 기능만큼 중요하다.**
   - Co-Star는 톤으로 기억된다.
   - The Pattern은 해석 프레임으로 기억된다.
   - Saajuu는 "받아주되 현실적으로 정리해주는 말투"가 핵심이다.

5. **관계형 서비스는 단순 광고보다 구독/상담/리텐션 매출이 강하다.**
   - 무료 트래픽만으로는 방어력이 약하다.
   - 관계 유지 서비스는 일상 사용성이 생기면 매출이 커진다.

### 3.2 주의할 것

1. 과한 자동결제 유도
2. 핵심 화면 안에서 상담 업셀을 과도하게 밀어 넣는 패턴
3. 치료나 전문가처럼 보이게 하는 표현
4. 카드만 뽑아주는 저품질 생성형 운세 UX
5. 사용자에게 이전 맥락을 기억하지 못하는 반복 답변

## 4. Saajuu에 바로 반영할 항목

### 4.1 무료 콘텐츠

우선순위 순서:

1. 결과 화면을 긴 스크롤 대신 탭/섹션 전환형으로 유지
2. `오늘의 한 줄 -> 세부 탭 -> 고민 연결 질문` 구조 강화
3. 인연 연결 뒤 `오늘의 관계 질문` 1개 제공
4. 궁합 결과 끝에서 `대화해볼 질문 3개` 제공
5. 월간/주간 카드도 긴 문단보다 카드형 요약 우선

### 4.2 상담 UX

반드시 반영:

1. 무료 3턴은 매 턴 역할이 달라야 함
   - 1턴: 마음 받기
   - 2턴: 조건 좁히기
   - 3턴: 행동 정리
2. 각 턴 시작 전에 직전 답변 1줄 요약 반영
3. "같은 답 반복" 방지를 위한 단계별 프롬프트/템플릿 분리
4. 무료 3턴 종료 후에는 유료 업셀보다 `요약 + 다음 질문`을 먼저 보여주기

### 4.3 인연 UX

바로 추가할 가치가 큰 것:

1. 연결된 상대와의 관계 카드
2. 카드에서 바로
   - 오늘의 관계 흐름
   - 궁합 보기
   - 상담 이어가기
3. 초대 수락 후 서로 다른 입력 순서여도 같은 궁합 결과 유지
4. 상대 표시명과 관계 설명을 관리하는 흐름 강화

### 4.4 수익화

초기 반영:

1. 광고보다 상담/리포트 중심
2. 구독은 나중, 지금은
   - 기본 상담권
   - 프로 상담
   - 관계/사업 리포트
3. 무료에서 유료로 넘어갈 때도 갑자기 결제시키지 말고
   - 지금까지 들은 고민 요약
   - 더 깊게 보면 무엇이 나오는지
   - 남는 결과물
   을 먼저 설명

## 5. 반영하지 않을 항목

아래는 지금 넣지 않는다.

| 항목 | 이유 |
|---|---|
| 치료/정신건강 앱 포지셔닝 | 법적 리스크와 톤 혼선 |
| 음성 다툼 녹음 분석 | 프라이버시 부담이 너무 큼 |
| 과한 커플 게임화 | Saajuu의 사주/상담 정체성과 어긋날 수 있음 |
| 실시간 전문가 마켓플레이스 | 운영 복잡도와 품질 관리 비용이 큼 |
| 공격적 구독 락인 | 초기 신뢰를 해침 |

## 6. 제품 결정

이 조사 기준으로 Saajuu의 다음 우선순위는 이렇게 정리한다.

1. **기억이 이어지는 무료 3턴 상담 고도화**
2. **인연 연결 뒤 재방문 가능한 관계 카드**
3. **긴 스크롤 대신 탭/카드 중심 정보 구조 유지**
4. **생활 언어 중심 해석 강화**
5. **유료 전환은 상담/리포트 명확화 후 진행**

즉, Saajuu는 "AI가 사주를 말해주는 앱"이 아니라 아래 쪽으로 더 선명해져야 한다.

> 사주를 근거로 관계와 결정을 정리해주는 한국형 AI 고민 상담 서비스
