# 점신 벤치마크 요약

작성일: 2026-07-11  
목적: GPT 챗에서 정리한 점신 실제 서비스 캡처 38장 기반 벤치마크를 Saajuu 개발 기준으로 고정한다.

## 원칙

- 점신의 화면, 문구, 아이콘, 브랜드, 레이아웃을 복제하지 않는다.
- 원본 캡처는 공개 저장소에 커밋하지 않는다. `점신/`, `local-reference/`는 로컬 참고 자료다.
- 벤치마크 대상은 기능 구조, 퍼널, 수익화 장치, 리텐션 장치다.

## 확인된 구조

점신은 단순한 무료 운세 앱이 아니라 다음 구조를 가진다.

```text
무료 운세·타로·심리 콘텐츠
  -> 매일 재방문과 프로필·인맥 축적
  -> 배너/전면 광고와 제휴 방문 보상
  -> 상담사 탐색과 코인 결제
  -> 부적·기원 상품 판매
```

## 핵심 화면군

| 영역 | 확인된 기능 |
|---|---|
| 운세보고서 | 오늘/내일/지정일 운세, 시간대별 흐름, 영역별 점수 |
| 인맥보고서 | 가족·연인·지인 등록, 궁합 순위, 관계 이벤트 |
| 행운보고서 | 음식, 코디, 야구, 골프, 재테크, 부동산 등 생활 추천 |
| 운세 허브 | 신년운세, 토정비결, 정통사주, 만세력, 관상, 심리풀이 |
| 타로 | 오늘/월간 타로, 연애·재회·재물·취업·건강·비즈니스 질문 |
| 참여형 도구 | 소원담벼락, 고민구슬, 포춘쿠키, 출석체크 |
| 상담 | 운세·타로·신점·심리상담, 첫 3분 무료, 상담사 필터 |
| 커머스 | 연애·금전·사업·건강·행운 부적 및 기원 상품 |
| 보상 | 행운패스, 복주머니, 제휴처 방문 보상 |

## 콘텐츠가 많아 보이는 원리

점신은 완전히 다른 엔진을 수십 개 운영하기보다, 사주/날짜/관계 데이터를 여러 표현 템플릿으로 재포장한다.

| 기본 데이터 | 재가공 콘텐츠 |
|---|---|
| 사주 명식 | 정통사주, 신년운세, 토정비결, 성향 풀이 |
| 일진·날짜 | 오늘, 내일, 지정일, 시간대별 운세 |
| 영역 점수 | 재물, 연애, 건강, 학업, 사업운 |
| 행운 요소 | 번호, 색상, 방향, 음식, 코디, 아이템 |
| 관계 정보 | 궁합, 인맥 순위, 추천 놀이·음식 |
| 고민 주제 | 타로 질문, 상담사 추천, 부적 상품 |

Saajuu도 `공통 운세 엔진 + 표현 템플릿 + 주제별 카드` 방식으로 확장한다.

## Saajuu가 가져올 것

- 오늘의 운세를 점수, 시간대, 분야별 해석, 행운 아이템으로 확장
- 인연/궁합을 핵심 리텐션 기능으로 설계
- 타로·심리·포춘쿠키 같은 낮은 진입 장벽 콘텐츠 추가
- 상담사 선택 전 신뢰 장치: 전문 분야, 스타일, 만족도, 후기 느낌의 요약
- 광고 보상은 선택형으로 두고, 기본 결과와 유료 상담 화면에는 방해하지 않음
- 상담/리포트/디지털 굿즈/선물하기를 수익화 후보로 관리

## 가져오지 않을 것

- 과도한 전면 광고
- 닫기 어려운 광고 경험
- 실제 인간 상담사처럼 보이게 하는 AI 표현
- 부적·액운 표현으로 불안을 과도하게 자극하는 문구
- 얼굴 인식 관상 기능의 초기 도입

## 차별화

| 점신 | Saajuu 방향 |
|---|---|
| 운세 콘텐츠 백화점 | 고민과 상담으로 이어지는 구조 |
| 인간 상담사 플랫폼 | 일관된 AI 페르소나와 공통 품질 기준 |
| 광고 중심 무료 경험 | 적은 광고와 깨끗한 상담 UX |
| 전화 상담 | 기록이 남는 텍스트 상담과 요약 리포트 |
| 결과 중심 | 선택지와 현실 행동 중심 |
