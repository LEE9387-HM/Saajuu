# 콘텐츠 품질 감사와 데이터 보강 계획

기준 버전: v0.5.22.0

## 현재 품질 진단

`manseryeok`은 생년월일시를 연주·월주·일주·시주로 변환하는 계산 기반이다. 현재 서비스의 오행 수, 십신, 일일 점수와 문장은 Saajuu가 별도로 만든 규칙과 템플릿이다. 따라서 명식 계산이 가능하다는 사실만으로 해석의 깊이나 상담 품질이 확보되지는 않는다.

현재 강점:

- 생년월일시에서 네 기둥을 일관되게 계산한다.
- 오행, 십신, 일진 관계를 구조화된 값으로 만든다.
- 해석 근거와 비단정적 행동 제안을 함께 보여준다.

현재 한계:

- 계절과 월령을 반영한 일간 강약 판단이 단순하다.
- 합·충·형·파·해, 지장간, 통근, 투간의 종합 우선순위가 없다.
- 대운 기산과 순행·역행, 세운·월운 교차 검증이 없다.
- 일일 점수는 전통 명리의 정답값이 아니라 Saajuu의 설명용 휴리스틱이다.
- 주제별 문장이 계산 규칙보다 템플릿 비중이 높아 비슷한 결과가 반복될 수 있다.
- 상담 LLM에는 명식의 구조화된 근거가 아직 직접 전달되지 않는다.

## 다음에 필요한 데이터

### 1. 검증된 명리 규칙 데이터

명리 전문가와 함께 아래 항목을 구조화된 규칙표로 만든다.

- 월령과 계절에 따른 일간 강약
- 지장간, 통근, 투간
- 천간합과 지지의 합·충·형·파·해
- 십신의 위치별 의미와 상호작용
- 조후와 오행 균형의 우선순위
- 대운 순행·역행과 기산 기준
- 세운·월운이 원국과 만날 때의 해석 우선순위

각 규칙에는 `근거 출처`, `적용 조건`, `충돌 시 우선순위`, `금지할 단정 표현`, `전문가 검수 상태`가 필요하다.

### 2. 주제별 상담 지식

사주 용어를 늘리는 것보다 실제 질문에 대응하는 상담 프레임이 우선이다.

- 연애: 확인된 행동, 기대, 거리감, 회복 방식
- 결혼: 돈, 가족, 생활 리듬, 책임 분담
- 사업: 고객 검증, 자금, 역할, 철수 기준
- 이직: 바꿀 수 있는 조건, 대안의 질, 전환 비용
- 가족: 반복 역할, 경계, 돌봄 부담, 대화 순서

각 주제는 `초기 질문`, `구체화 질문`, `선택지 비교`, `다음 행동`의 4단계 템플릿으로 관리한다.

### 3. 평가용 사례 세트

실제 사용자 개인정보가 아닌 합성 사례로 최소 100개 평가 세트를 만든다.

- 같은 명식, 다른 고민일 때 답변이 달라지는지
- 두 사람의 입력 순서를 바꿔도 궁합 점수가 같은지
- 상담 1·2·3회 답변이 각각 다른 역할을 하는지
- 근거 없는 예언, 불안 자극, 의료·법률·투자 단정이 없는지
- 계산 근거와 최종 문장이 모순되지 않는지

## 권장 구축 순서

1. 현재 규칙을 `계산 사실`, `해석 규칙`, `표현 문장`으로 분리한다.
2. 사업·연애·결혼 세 주제부터 전문가 검수 규칙표를 만든다.
3. Edge Function에 생년월일 원문 대신 구조화된 사주 요약을 전달한다.
4. 합성 사례 100개로 회귀 평가를 자동화한다.
5. 평가 통과 후 대운·세운·월운을 프로 상담에만 단계적으로 연다.

## 제품 표시 원칙

- 사용자 화면에는 경쟁 서비스명, 개발 단계, 내부 데이터명, 업셀 계획을 노출하지 않는다.
- 계산 근거는 원하는 사용자만 `계산 근거` 탭에서 본다.
- 무료 결과의 첫 화면은 사용자가 적은 질문과 직접 연결한다.
- 일일 점수는 절대적인 길흉 점수가 아니라 그날의 상대적 리듬임을 유지한다.
- 검수되지 않은 고급 명리 용어를 콘텐츠 양을 늘리기 위해 추가하지 않는다.
