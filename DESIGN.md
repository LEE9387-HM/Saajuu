# Saajuu 디자인 시스템

이 문서가 UI 결정의 기준이다. 새 화면·컴포넌트는 여기 어휘로 먼저 조립하고, 없을 때만 추가한다.

## 무드

한지 위에 먹으로 눌러 쓴 작은 기록. 차분하고, 종이 같고, 예언서가 아니라 노트처럼 읽힌다.
화려함 대신 여백과 가는 선(hairline border)으로 격을 만든다.

## 색 토큰 (styles.css `:root`)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--paper` (배경) | `#fbf6ec` | 페이지 배경, 크림지 |
| 카드 배경 | `#fffdf8` | 모든 카드/폼 표면 |
| `--ink` | `#152a38` | 본문·제목, 먹색 |
| `--muted` | `#718087` | 보조 텍스트, 날짜, 힌트 |
| `--jade` | `#4d8072` | 긍정 포인트: 가까이할 것, 행운 요소, 포커스 링 |
| `--gold` | `#c5964f` | 강조 프레임, 티저 라벨, 카드 저장 버튼 |
| `--line` | `#ddd3c0` | hairline 보더 |
| 경고 | `#b0563f` | 조심할 것, 피해야 할 해석 |
| 본문 보조 | `#53646b` | 카드 본문 문단 |

규칙: 보라·파랑 그라데이션 금지. 배경은 항상 크림지. 액센트는 옥(jade)·금(gold) 둘뿐.

## 타이포그래피

- 서체: "Gowun Batang" 세리프 (제목·본문 공통). 기본 스택(Inter/Roboto/system) 금지
- eyebrow 라벨: 11px 내외, `letter-spacing: 0.35em`, 주홍 `#c05a44` 또는 gold
- 한 줄 결론(verdict): 카드에서 가장 큰 텍스트. 1.7~2rem, weight 700, 부정문 금지, 25자 내외
- 근거 칩(evidence): 11px, hairline 보더, muted — 계산값을 담백하게

## 컴포넌트 어휘

- **카드**: `#fffdf8` 배경 + `--line` 1px 보더. 라운드는 4px 이하(각진 종이 느낌). 그림자 없음
- **강조 카드**: 보더를 `rgb(197 150 79 / 45%)`(옅은 금)로 — 처방·일일 운세 등 "오늘 볼 것"
- **2열 처방**: `.guidance__col--embrace`(jade 제목) / `--avoid`(경고색 제목)
- **인용 경고 블록**: 좌측 3px 경고색 보더 + `#f4ede0` 배경
- **evidence 칩 행**: 모든 해석 카드 하단에 계산 근거 칩 필수 — 이 앱의 신뢰 장치
- **버튼**: 먹색 배경 기본, jade(신청)·gold(저장) 변형. 라운드 4px

## 글쓰기 규칙 (카피)

1. 모든 해석 카드의 첫 줄은 한 줄 결론 — "무엇을 하라/조심하라"가 보여야 한다
2. "~가 아닙니다"로 시작하는 방어적 문장 금지 — 무엇인지 먼저, 한계는 뒤에
3. 단정 예언 금지: "반드시 ~된다" 대신 "~하기 좋은 날/흐름"
4. 행동으로 닫기: 카드마다 이번 주/오늘 할 수 있는 행동 하나

## 접근성

- 색 정보(행운 색 등)는 스와치 + 색 이름 텍스트 병기 — 색 단독 전달 금지
- 본문 16px 이상, 대비 4.5:1 이상
- 터치 타깃 44px 이상, 갱신되는 영역은 `aria-live="polite"`

## 금지 패턴 (AI 슬롭 가드)

아이콘 원형 배지 3열 그리드, 이모지 장식, 물결 SVG 디바이더, 떠다니는 블롭,
전부 중앙 정렬, 카드 좌측 컬러 보더 장식, "당신의 올인원 솔루션" 류 카피.
섹션이 비어 보이면 장식이 아니라 콘텐츠를 고칠 것.
