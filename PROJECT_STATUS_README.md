## 제품 방향

Saajuu는 브라우저 안에서 사주 계산과 해석을 제공하고, 결과를 오늘 운세, 연간 운세, 궁합, 타로, AI 상담으로 이어 주는 서비스입니다.

핵심 방향은 다음과 같습니다.

1. 무료 콘텐츠만으로도 기본 만족감을 준다.
2. 결과가 길기만 한 운세가 아니라, 실제 고민과 다음 행동으로 이어지게 만든다.
3. 로그인 이후에는 인연 연결, 상담 기록, 결제형 상담으로 자연스럽게 확장한다.

## 일일 운세 알고리즘

일일 운세는 서버 호출 없이 브라우저 안에서 생성한다.

기본 원칙은 다음과 같다.

1. 입력된 사주 차트에서 오행 균형, 일주, 십신 등 핵심 신호를 뽑는다.
2. 현재 날짜를 기준으로 일일 키를 만든다.
3. 사주 신호와 날짜 키를 섞어 매일 바뀌는 점수와 문장 조합을 만든다.
4. 총점, 세부 영역 점수, 한 줄 요약, 행동 제안, 상담 연결 질문까지 같이 생성한다.

구현 위치:

- [src/daily.js](C:\WorkSpace\메모\10. Project\Saajuu\src\daily.js)
- 일부 해석 조립은 [src/fortune.js](C:\WorkSpace\메모\10. Project\Saajuu\src\fortune.js)에서 재사용

## 한자 성명학 데이터 파이프라인

현재 한자 성명학은 브라우저에서 동작 가능한 정적 데이터 기반 구조다.

구성은 다음과 같다.

1. 이름 입력
2. 한글 이름 정규화
3. 후보 한자 데이터 로드
4. 선택된 한자 기준 획수/오행/보조 해석 계산
5. 결과를 사주 본문과 별도 보조 리딩으로 노출

- 실제 LLM 응답 품질은 서버 프롬프트와 모델 설정 영향이 크므로 지속 보정 대상

### 결제

- PortOne 브라우저 SDK 연결
- 결제/복구/상담권 관리의 기본 운영 경로 존재
- 운영 비밀키와 웹훅 시크릿은 Edge Function Secret으로 관리

## 안전 / 법적 가드레일

- 생년월일시를 정적 공유 링크에 직접 넣지 않는다.
- 브라우저 코드에 API 키, 비밀키, LLM 키를 넣지 않는다.
- 의료, 투자, 법률, 임신, 생사처럼 단정적이거나 위험한 예언형 표현을 쓰지 않는다.
- 상담 결과는 한 줄 결론 뒤에 근거와 다음 행동이 이어지는 형태로 쓴다.
- 경쟁 서비스의 문구/레이아웃/브랜드를 복제하지 않는다.
