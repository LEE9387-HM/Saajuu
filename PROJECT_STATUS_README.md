# Saajuu 프로젝트 현황 / 핸드오프 문서

## v0.5.44.0 상담 요약/단계 맥락 복원 (2026-07-19)

- 상담 메시지를 보낼 때마다 `session_summaries`에 현재까지의 고민 요약, 다음 단계 안내, 행동 프롬프트를 upsert 하도록 연결했습니다.
- 이제 `마이 -> 상담 이어가기`나 같은 상담권 재진입 시 이전 메시지뿐 아니라 현재 상담이 무엇을 정리하고 있었는지도 함께 복원됩니다.
- `activeConsultationGuidance`가 저장된 요약 데이터를 다시 읽어 오기 때문에, 재진입 직후 안내 칩과 입력 프롬프트가 빈 상태로 시작하지 않습니다.
- 검증: `npm test`, `npm run build`

## v0.5.43.0 상담 재진입 시 이전 메시지 복원 (2026-07-19)

- 무료 3턴이든 유료 상담권이든, 이미 열린 세션을 다시 시작하면 `consultation_messages`를 읽어와 이전 대화가 그대로 보이게 연결했습니다.
- 이제 `마이 -> 상담 이어가기`나 같은 상담권 재진입 시 대화창이 비어 보이지 않고, 사용자가 직전에 어디까지 이야기했는지 바로 이어서 볼 수 있습니다.
- 검증: `npm test`, `npm run build`

## v0.5.42.0 유료 상담권 세션 시작 연결 (2026-07-19)

- 무료 3턴 흐름과 별도로, 활성 이용권이 있는 사용자는 같은 상담 화면에서 바로 `기본 상담` 또는 `프로 상담` 세션을 시작할 수 있게 연결했습니다.
- `마이` 화면의 이용권/주문 카드에서 선택한 entitlement ID를 상담 진입까지 유지해, 사용자가 방금 산 상담권으로 곧바로 이어서 들어가도록 맞췄습니다.
- 결제 완료 직후 새 entitlement를 다시 조회하고 해당 상담 모드와 이용권을 자동 선택해 상담 섹션으로 복귀시키도록 보강했습니다.
- `create-consultation-session`, `send-consultation-message` Edge Function을 함께 확장해 유료 세션 재진입과 턴 차감이 실제 DB 상태와 일치하도록 맞췄습니다.
- 검증: `npm test`, `npm run build`

## v0.5.41.0 마이 페이지에서 상담 재진입 연결 (2026-07-19)

- 결제 실패·취소 주문도 같은 상품/상담사 조합으로 다시 준비할 수 있게 해서 `마이 -> 상담` 재진입 흐름을 정리했습니다.
- 검증: `npm test`, `npm run build`

- `연간·월간 운세` 카드가 더 이상 플레이스홀더가 아니라, 입력한 사주를 기준으로 월별 점수 12개, 상반기/하반기 요약, 이번 달 집중 포인트를 보여주는 실제 무료 콘텐츠로 연결되었습니다.

- 이번 변경으로 무료 허브의 빈 카드 두 장을 실제 콘텐츠로 대체했고, 애드센스나 보상형 광고를 붙이기 전에 필요한 기본 체류 콘텐츠 밀도를 한 단계 올렸습니다.
