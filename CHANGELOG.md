# Changelog

## 0.5.59.0 - 2026-07-22

- 연간·월간 운세 카드에 `강한 구간 / 속도 조절 구간 / 올해 운영 원칙` 해석 블록을 추가해 기본 콘텐츠 문량을 늘렸습니다.
- 월별 칩마다 한 줄 풀이 외에 `다음 행동` 문장을 추가해 점수형 결과가 실제 행동 질문으로 이어지도록 보강했습니다.
- 타로 카드에 `짧은 판단 / 왜 이렇게 읽는가 / 지금 할 한 가지` 해석 카드를 추가하고, 정리 질문과 후속 질문 수를 늘렸습니다.
- 검증: `npm test`, `npm run build`

## 0.5.58.0 - 2026-07-22

- 같은 인연 카드에서 방금 본 궁합 결과가 있으면, 그 한 줄 요약과 관계 흐름, 다음 행동이 상담 시작 문장에 자동으로 반영되도록 보강했습니다.
- 다른 관계를 새로 선택하면 이전 궁합 결과가 섞이지 않도록 궁합 컨텍스트와 상담 시작 문장을 함께 초기화하도록 정리했습니다.
- `PROJECT_STATUS_README.md`와 `CHANGELOG.md`를 현재 상태 기준의 정상 한글 문서로 다시 정리했습니다.
- 검증: `npm test`, `npm run build`

## 0.5.57.0 - 2026-07-22

- 인연 카드에서 `이 관계로 궁합 보기`를 누르면 선택된 관계 컨텍스트가 유지되도록 관계 배너와 선택 상태를 추가했습니다.
- 궁합 결과와 선택된 관계를 연결해, 같은 관계 맥락으로 바로 상담을 시작할 수 있는 CTA를 추가했습니다.
- 연결된 관계가 사라지거나 선택이 해제되면 궁합 컨텍스트도 함께 정리되도록 상태 흐름을 보강했습니다.
- 검증: `npm test`, `npm run build`

## 0.5.56.0 - 2026-07-21

- 궁합 결과 카드에 `관계 흐름`, `이번에 해볼 한 가지`, `더 좁혀볼 질문` 블록을 추가해 점수형 결과가 실제 대화와 상담 흐름으로 이어지도록 보강했습니다.
- 연인/부부/재회/가족 관계별로 궁합 후속 질문, 다음 행동, 상담 연결 질문이 다르게 나오도록 관계별 프레임을 추가했습니다.
- 궁합 결과 영역의 깨진 문구를 정상 한글로 정리하고, 관련 테스트를 확장해 관계별 안내 문장이 다시 단순 점수형으로 축소되지 않도록 잠갔습니다.
- 검증: `npm test`, `npm run build`

## 0.5.55.0 - 2026-07-21

- 타로 무료 콘텐츠를 핵심 카드, 보조 카드, 경계 카드의 3카드 구조로 확장했습니다.
- 사업, 연애, 결혼, 직업, 가족, 일반 주제에 따라 타로 요약과 후속 질문이 다르게 나오도록 보강했습니다.
- 타로 카드 아래에 더 좁혀볼 질문과 마무리 문장을 추가해 실제 고민 정리 흐름으로 이어지게 만들었습니다.
- 검증: `npm test`, `npm run build`

## 0.5.54.0 - 2026-07-21

- 연간·월간 운세 해석이 사업, 연애, 결혼, 직업, 가족, 일반 주제에 따라 다른 문장 프레임을 타도록 확장했습니다.

- 검증: `npm test`, `npm run build`
