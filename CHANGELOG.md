# Changelog

이 프로젝트의 주요 변경사항을 기록합니다.

## [0.5.1.0] - 2026-07-11

### Added

- GPT 챗 수익화 계획 반영 문서 세트 추가: 점신 벤치마크, 무료 콘텐츠 명세, AI 페르소나/프로 모드, 상담 퍼널, 서버·개인정보·결제 기준, 구현 로드맵
- 모바일 정보구조: 오늘 / 내 사주 / 올해 / 인연 / 타로 / 상담 / 마이 앵커 내비게이션
- 무료 콘텐츠 허브: 연간·월간 운세, 1회성 인연·궁합, 타로·테스트, AI 상담, 마이 영역의 초기 안내 카드
- 저장된 사주가 없을 때 오늘의 운세 영역에서 입력을 유도하는 빈 상태

### Changed

- `AGENTS.md` 필수 읽기 목록에 신규 제품 기준 문서 추가
- 점신 캡처, 로컬 참고 자료, NVIDIA Persona 원본, 시크릿 파일이 커밋되지 않도록 `.gitignore` 확장

## [0.5.0.0] - 2026-07-11

### Added

- 한자 성명학 정밀 풀이: 대법원 인명용 한자(약 9,500자)로 이름 각 글자의 한자를 골라 원형이정(元亨利貞) 사격과 81수리 길흉을 계산
- 원획(강희자전 기준) 계산: 강희 214부수 본자 획수 + 나머지 획수. 부수 변형(氵→水 등) 자동 보정, 숫자 한자는 수의(數意) 획으로 처리
- 복성(두 글자 성)·외자 이름 대응, 유파 차이 고지 문구
- 한자 데이터(`src/data/name-hanja.json`)는 버튼 클릭 시 lazy-load하는 별도 청크로 분리
- 변환 스크립트 `scripts/build-hanja-db.mjs`: krcourt SQLite(webhanja.db) → JSON. 데이터는 저작권법 제7조에 따라 보호 대상 아님

### Note

- 81수리 길흉 분류는 유파에 따라 차이가 있어 참고용으로 제공

## [0.4.1.0] - 2026-07-10

### Added

- 결과 링크 공유: 생년월일시·달력·주제만 해시(`#r=...`)로 인코딩 — 이름·고민·톤은 절대 미포함. Web Share 지원 기기는 공유 시트, 그 외 클립보드 복사
- 공유 링크 열람: 일회성 렌더로 저장 프로필을 덮지 않음, "내 사주로 돌아가기" 버튼, 같은 탭 해시 변경(hashchange) 처리, 렌더 후 해시 자동 제거
- GoatCounter 사이트 코드(fortune9388) 활성화 + share-link/share-open 이벤트

### Changed

- 손상된 공유 링크는 오류 안내 후 일반 흐름으로 폴백

## [0.4.0.0] - 2026-07-10

### Added

- 오늘의 운세: 일진(60갑자)과 원국을 대조한 매일 다른 개인화 운세 — 한 줄 결론, 조심할 것, 행운의 숫자·색(십신 10종 × 지지 관계 5분류, 우선순위 충>형>육합>반합)
- 내일 운세 티저로 재방문 동기 부여
- 프로필 기기 내 저장(localStorage v1 스키마): 재방문 시 입력 없이 오늘의 운세를 폼보다 먼저 표시, 정보 수정·삭제 버튼 제공
- GoatCounter 계측(미설정 시 no-op): 방문 + premium-interest / card-save / daily-view 이벤트
- 모든 주제 카드에 한 줄 결론(verdict) 우선 표기
- DESIGN.md 디자인 시스템 문서

### Changed

- "~가 아닙니다"로 시작하던 방어적 카피 전면 재작성 (결혼·사업·가족·신년)
- 프리미엄 사전 신청 버튼의 영구 비활성화 제거 — 수요 이벤트 상시 계측
- 사주 카드 저장 시 화면 렌더 시점 값 재사용 (자정 직후 불일치 방지)

## [0.3.0.0] - 2026-07-08

### Added

- 올해의 흐름과 처방: 세운(올해 간지)과 원국을 대조해 "가까이할 것 / 조심할 것"을 제안하는 점술형 처방 섹션
- 나의 사주 카드 저장: 네 기둥과 올해의 처방을 담은 카드 이미지를 저장·공유하는 기능
- 유료 상담 사전 신청 버튼(오픈 알림 신청)으로 수요 신호 수집

### Changed

- 결과 화면에서 내부 개발 계획(서버/API/로드맵) 문구를 제거하고 사용자용 안내로 교체

### Removed

- Now/Next/Paid 고도화 아키텍처 카드와 백엔드 설명 섹션 (기획 문서로 이동)
