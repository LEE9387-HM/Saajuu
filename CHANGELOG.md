- 출생시각의 분을 0~59 사이에서 정확하게 입력하는 기능
- 일간·오행·십신 계산 근거와 자기 관찰 질문을 함께 제공하는 상세 풀이

- 계산 로직 자동 테스트와 GitHub Pages 자동 배포 워크플로
